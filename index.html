Privacy Policy

Effective: April 1, 2025 · Updated: March 2026

Introduction

D Lingo respects your privacy. No personal data is stored on our servers.

What We Collect

  • Microphone input — only while you press mic
  • Speech-to-text — via browser's Web Speech API
  • Room messages — stored temporarily in Firebase; auto-deleted after session
  • Language preferences — browser memory only

Data Storage

  • No permanent voice recordings
  • Room messages auto-expire; no permanent profiles
  • No login, no accounts

Additional Privacy Notes

  • Voice data is processed temporarily and not stored permanently.
  • Conversations in live rooms are not retained after session ends.
  • No personal identification data is collected by D Lingo.

Third-Party Services

  • Web Speech API — Chrome's built-in recognition
  • MyMemory Translation API — by Translated.net
  • Firebase Realtime DB — Google Firebase for room messaging
